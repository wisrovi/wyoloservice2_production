The Intelligent Stack

Modular Neural Architecture

Five specialized components working in perfect harmony to deliver enterprise AI training.

The Interface

NeuroForge UI

React/Vite dashboard. Real-time telemetry, terminal widgets, and an AI Assistant powered by Gemini.

The Gateway

FastAPI Server

High-performance REST API. Handles task enqueuing, priority management, and metrics aggregation.

The Brain

Orchestrator

Managing Optuna studies and genetic hyperparameter search. Decides WHAT and WHEN to train.

The Hive

GPU Invoker

Distributed Celery workers listening for jobs. Handles GPU allocation and dataset mounting via Samba.

The Muscle

YOLO Executor

GPU-isolated process running pure YOLOv8/v11 training. Communicates results back to MLflow and Redis.

Infrastructure

Data Backbone & Communication

Built on top of industry-standard tools for maximum reliability and scalability.

Redis & Celery

Asynchronous task distribution. Low-latency broker for cross-component communication and job queues.

MLflow & MinIO

Full lifecycle tracking. S3-compatible storage for datasets and model weights with centralized metric logging.

PostgreSQL

Relational persistence for user registries, project configurations, and Optuna optimization histories.

Samba (CIFS)

Seamless data sharing. Mount training datasets from the central server to remote worker nodes instantly.

Watchtower

Zero-downtime updates. Automatically monitors and updates all system containers to the latest versions.

Flower

Comprehensive monitoring for the Celery cluster. Track task health, worker load, and queue depth.
