# Roadmap - WDarwin Ops Production

## Phase 1: High Availability
- **Postgres Replication**: Master-Slave setup for zero-loss metadata.
- **Redis Sentinel**: High-availability broker cluster.

## Phase 2: Cloud Native
- **Kubernetes Support**: Helm charts for one-click cluster deployment.
- **Terraform**: Infra-as-code for AWS/GCP GPU instances.
